# Sample: Welcome to the Blog

This is a **sample post** in Markdown. To add a new post:

1. Add a `.zh.md` and/or `.en.md` file under `blog/posts/` (e.g. `2025-03-my-post.zh.md`).
2. Add an entry in `blog/index.json` with `slug`, `title_zh`, `title_en`, and `date`.
3. Push to GitHub; the list and posts will update automatically.

## Markdown

- **Bold**, *italic*
- Lists, blockquotes, code blocks
- [Links](https://example.com), images
