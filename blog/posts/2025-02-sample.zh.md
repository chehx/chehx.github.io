# 示例：欢迎来到博客

这是一篇**示例文章**，用 Markdown 写成。你之后只需要：

1. 在 `blog/posts/` 下新建 `.zh.md` 和/或 `.en.md` 文件（如 `2025-03-我的第一篇.zh.md`）；
2. 在 `blog/index.json` 里增加一条记录（`slug`、`title_zh`、`title_en`、`date`）；
3. 保存并推送到 GitHub，博客列表和文章页会自动更新。

## 支持的 Markdown 语法

- **粗体**、*斜体*
- 列表、引用、代码块
- [链接](https://example.com)、图片

```python
# 代码块也会正确渲染
def hello():
    print("你好，博客")
```

使用页头的语言切换（中文 | English）可切换界面与文章语言。
